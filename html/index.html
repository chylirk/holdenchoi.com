Hi, my name is Holden.

I'm a software engineer based in Austin, TX.

I recently built Retrospect, an open-source observability tool that allows developers to easily debug in a distributed system.

Retrospect is an open-source observability tool for distributed back-end tracing and front-end session replay. It allows developers to easily debug their applications by abstracting away the complexity of tracing instrumentation and session recording.
